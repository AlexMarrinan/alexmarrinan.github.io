Gotta Go Bot

Itch Page

Gotta Go Bot! is a face paced skill based 3D platformer developed in Unreal Engine 5. You play as a small robot exploring the remains of a much larger robot. Use your dash and graple abilities to traverse this abandoned, dangerous, overgrown, yet beutfiul enviroments.

This project was devloped for the class IMGD 4000: Technical Development II. My role was as lead gameplay programmer. Myself and Jade McAvoy, a fellow programmer on the team, designed the movment and character control. I implemented all code related to character movment, including basic movement, jumping, grappeling, and dashing. I also hocked up art assets and animations to react based on player input and the enviorment.
